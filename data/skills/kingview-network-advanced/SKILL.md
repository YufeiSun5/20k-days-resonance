# 组态王：网络与高级功能

本 Skill 提供组态王 7.5 **网络功能、OPC、冗余、Web发布及其他高级功能**的完整参考，涵盖第二十章至第三十五章（不含驱动章节）。

## 适用场景

- **OPC 设备**（第20章）：配置组态王作为 OPC 客户端，连接 OPC 服务器
- **网络功能**（第21章）：C/S 架构配置，服务器/客户端节点设置，网络变量
- **冗余系统**（第22章）：主/备服务器冗余切换配置
- **Web 发布**（第23章）：将组态王画面发布为 Web 页面，IE 浏览器访问
- **历史库高级**（第24章：见 `kingview-database`）
- **第三方程序访问**（第25章）：通过 DDE/OPC/SDK 让其他程序读写组态王数据
- **XML 导入导出**（第26章）：工程数据的 XML 格式导出与导入
- **模板**（第27章）：画面模板和变量模板的创建与使用
- **移动客户端**（第28章）：手机/平板访问组态王画面
- **电子签名**（第29章）：操作记录与电子签名合规配置
- **工具**（第30章）：文本转换、画面转换等实用工具
- **批次管理**（第31章）：批次记录、批次报表
- **多语言组态**（第32章）：工程多语言切换配置
- **KingHtmlBridge 控件**（第33章）：嵌入 HTML 页面
- **WebService**（第34章）：调用外部 WebService 接口
- **HTTP Server**（第35章）：组态王内置 HTTP 服务器功能
