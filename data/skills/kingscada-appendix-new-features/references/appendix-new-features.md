# 附录三 版本新增功能

KingSCADA3.8版本较KingSCADA3.7SP1版本新增加功能清单列表：

 

|  |  |
| --- | --- |
| 序号 | 新增功能描述 |
| 1 | KSP支持KS冗余功能 |
| 2 | KSP支持变量域的读写功能 |
| 3 | IO变量写操作结果回馈功能 |
| 4 | Web发布支持变量域的发布策略 |
| 5 | 支持添加远程ks服务器/IOServer站点功能 |
| 6 | 增加获取当前客户端授权占用个数 |
| 7 | 新增冗余状态系统变量值±3 |
| 8 | 数据词典导入变量带变量分组和报警组功能 |
| 9 | 报警文本长度扩展到128个 |
| 10 | 支持国产深思锁功能 |
| 11 | 增加报表单元格统计函数的功能 |
| 12 | 在报表中生成条形码，可通过报表方法实现 |
| 13 | 企业微信转发增加报警文本功能 |
| 14 | 企业微信报警记录进行分级转发 |
| 15 | 新增方法用户登出时无需弹窗 |
| 16 | 系统的自定义脚本支持分组功能 |
| 17 | 通过引用变量进行同步写设备的函数 |
| 18 | 新增删除文件夹和文件的系统函数 |
| 19 | server系统和win10系统微软输入法冲突的处理 |
| 20 | KS数据词典中快速添加变量带kio中的变量组 |
| 21 | 通用历史趋势曲线控件设置 |
| 22 | 曲线游标新增加属性选择可以配置 |
| 23 | XY曲线横坐标支持脚本中修改点数 |
| 24 | 修改XY曲线控件需放开最大点数设置 |
| 25 | scada主服务器启动“登录访问数据”模式权限 |
