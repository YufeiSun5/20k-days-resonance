# 组态王：图形画面与动画连接

本 Skill 提供组态王 7.5 **图形画面设计和动画连接**的完整参考，涵盖第七章、第十三章（图库）、第十四章（控件）。

## 适用场景

- 新建画面、设置画面属性（大小、刷新率、背景色）
- 绘制图形元素：线、矩形、圆、文本、位图等
- 配置**动画连接**：
  - 填充、线条颜色、闪烁（显示/隐藏）
  - 水平/垂直位置移动、大小变化
  - 模拟值/离散值/字符串输出显示
  - 按钮（弹起/按下命令）、滑动杆输入
- 使用**图库向导**插入标准工控图形（阀门、泵、罐、电机等）
- 在画面中嵌入 **ActiveX 控件**
- 画面之间的切换（ShowPicture 等命令）

## 实战坑点与案例补充

### ActiveX 曲线画面的稳定组织方式

- 当一个主画面需要在“温度/压力/其他”与“历史/实时”之间切换时，优先考虑“主画面 + 覆盖式子画面”的结构，而不是在同一个 ActiveX 控件上反复动态改绑定。
- 现场验证过的稳定方案是拆成 `6` 个子画面：
  - `曲线_历史_温度`
  - `曲线_历史_压力`
  - `曲线_历史_其他`
  - `曲线_实时_温度`
  - `曲线_实时_压力`
  - `曲线_实时_其他`
- 切换时统一先 `HidePicture()`，再按 `iMode` 和 `iTab` 调 `ShowPicture()`。这种方式比“在一个控件里临时切变量、临时切模式”更容易维护，也更适合 ActiveX 控件。

### 典型切换脚本

```c
HidePicture("曲线_历史_温度");
HidePicture("曲线_历史_压力");
HidePicture("曲线_历史_其他");
HidePicture("曲线_实时_温度");
HidePicture("曲线_实时_压力");
HidePicture("曲线_实时_其他");

IF (iMode==0 && iTab==0) { ShowPicture("曲线_历史_温度"); }
IF (iMode==0 && iTab==1) { ShowPicture("曲线_历史_压力"); }
IF (iMode==0 && iTab==2) { ShowPicture("曲线_历史_其他"); }
IF (iMode==1 && iTab==0) { ShowPicture("曲线_实时_温度"); }
IF (iMode==1 && iTab==1) { ShowPicture("曲线_实时_压力"); }
IF (iMode==1 && iTab==2) { ShowPicture("曲线_实时_其他"); }
```

### 日期时间输入控件建议

- 历史查询界面如果需要“日期选择器 + 时间选择器”，优先插入 `Microsoft Date and Time Picker Control`。
- 若工程环境里没有该控件，退化方案是用数值输入框或字符串输入框分别输入年月日时分。
- `DTPicker` 的事件页常见事件为 `SelChangeDate`，但多数查询场景不必在事件里同步变量；通常在查询按钮里直接读控件属性更简单。
