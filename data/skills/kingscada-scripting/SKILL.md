# KingScada 第11章：脚本

本 Skill 提供北京亚控 KingScada SCADA 系统中**命令语言脚本**的完整参考，包括语法规则、数据类型、事件脚本分类和编辑器使用说明。

## 适用场景

- 查询脚本支持的**数据类型**（int、float、string、bool 等）和**运算符**
- 查询支持的**脚本语句**（if/else、while、for、switch 等控制流语法）
- 区分**全局事件脚本**与**局部事件脚本**的使用场景
- 使用**脚本编辑器**（智能提示、语法加亮、快捷键）
- **导入/导出/复制**脚本，以及自定义脚本分组
- 理解脚本函数的调用方式

## 内容结构

完整文档位于 [./references/ch11-scripting.md](./references/ch11-scripting.md)：

| 章节 | 内容 |
|------|------|
| 11.1 概述 | 脚本总体介绍 |
| 11.2 脚本编辑器 | 菜单栏、关键字区、语法加亮、智能提示、快捷键 |
| 11.3 脚本语法 | 支持的数据类型、运算符、脚本语句（控制流） |
| 11.4 脚本分类 | 全局事件脚本 vs 局部事件脚本 |
| 11.5 导入导出/复制/分组 | 脚本管理功能 |
| 11.6 脚本函数 | 在脚本中调用内置函数 |

## 使用方法

1. **加载参考文档**：读取 `./references/ch11-scripting.md`
2. **定位章节**：按关键词匹配（语法问题找 11.3，事件类型找 11.4）
3. **结合函数手册**：脚本中具体函数的签名和参数说明请参考 `kingscada-functions` Skill
