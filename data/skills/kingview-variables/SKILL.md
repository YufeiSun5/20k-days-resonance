# 组态王：变量定义与管理

本 Skill 提供组态王 7.5 **变量定义与管理**（数据词典）的完整参考，对应第五章。

## 适用场景

- 定义**内存变量**（离散、整数、实数、字符串）
- 定义**I/O变量**并关联 I/O 设备与寄存器
- 配置变量的**基本属性**（类型、初始值、工程单位、报警定义、记录/安全）
- 管理**变量数据库**（数据词典）：查找、修改、批量导入导出
- 了解**系统预定义变量**（\\$日期、\\$时间、\\$用户名等）
