# 附录二 常见问题解决方法

使用本 skill 回答 KingSCADA 中与“附录二 常见问题解决方法”相关的问题。

## 参考资料

完整资料位于 [./references/appendix-faq.md](./references/appendix-faq.md)。

## 使用流程

1. 先读取参考资料，按用户关键词定位相关标题、函数名、属性名或配置项。
2. 回答时优先给出可执行步骤、配置路径、参数含义和注意事项。
3. 涉及脚本或函数调用时，保留原手册中的函数名、参数顺序和返回值约束。
4. 如果手册内容不足以确定结论，明确说明缺口，并给出需要在 KingSCADA 工程中验证的点。

## 输出要求

- 不要整段复述手册；只提炼与用户问题直接相关的步骤和限制。
- 对配置类问题，按“位置 -> 操作 -> 验证”组织答案。
- 对脚本类问题，给出最小可用示例，并说明变量、控件或数据源需要按现场工程替换。
